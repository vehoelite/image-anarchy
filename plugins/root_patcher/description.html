üì± Device Mode (NEW!)

Connect your phone directly via USB and let the plugin handle everything automatically:

üöÄ Automated Root Process

  1. Scan Devices - Detects phones connected via ADB or Fastboot
  2. Extract boot.img - Pulls boot image directly from device (requires root)
  3. Patch on Device - Uses Magisk app on phone to patch the boot image
  4. Flash Back - Flashes patched boot via fastboot automatically

üìã Requirements for Device Mode

  • USB Debugging enabled on your device
  • Platform Tools (ADB/Fastboot) installed
  • Root access on device (to extract boot.img) - OR provide boot.img manually
  • Magisk app installed (for patching)
  • Unlocked bootloader (for flashing)

üìÑ File Mode

Traditional mode - analyze boot images and get patching instructions:

  1. Select your boot.img file
  2. Click Analyze to check compatibility
  3. Review the compatibility results
  4. Select your preferred rooting method
  5. Follow the provided instructions to patch
  6. Flash the patched boot image using fastboot or mtkclient
